# 当前构建时的日志显示

- 展示一大堆日志，很多并不需要开发者关注
- 需要`friendly-errors-webpack-plugin`， 不过下载不成功，就没整了，所以就没有想要的标色效果

## 统计信息 stats

| Preset        | Alternative | Description                    |
| ------------- | ----------- | ------------------------------ |
| "errors-only" | none        | 只在发生错误时输出             |
| "minimal"     | none        | 只在发生错误或有新的编译时输出 |
| ”none“        | false       | 没有输出                       |
| “normal"      | true        | 标准输出                       |
| "verbose"     | none        | 全部输出                       |
