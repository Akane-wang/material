# 充分利用缓存提升二次构建速度

- 目的：提升二次构建速度

- 缓存思路：

  - babel-loader开启缓存

    ```js
    module: {
    	rules: [
            {
                test: /\.js$/,
                use: [
                    'babel-loader?cacheDirectory=true', // babel-loader开启缓存
                ]
            }
        ]    
    }
    ```

    

  - cache开启缓存

    ```js
    module.exports = {
        cache: true, // boolean | object,生产环境默认为false
    }
    ```

    

  - 使用cache-laoder 或hard-source-webpack-plugin
