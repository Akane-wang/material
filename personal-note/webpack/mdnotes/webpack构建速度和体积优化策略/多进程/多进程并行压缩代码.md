# 多进程并行压缩代码
- 多进程/多实例：并行压缩

  - 使用parallel-uglify-plugin插件

  - uglifyjs-webpack-plugin开启parallel参数

    - 不支持es6语法压缩

  - terser-webpack-plugin开启parallel参数

    - 支持es6语法压缩

      ```js
      module.exports = {
          optimization: {
              minimizer: [
                  new TerserPlugin({
                      parallel: 4 // 默认是cpu数量*2-1 | true 即为默认值
                  })
              ]
          }
      }
      ```
